### 学习技巧

**把握本质**

+ JavaScript 中引用类型的值是按 **引用访问的**
+ JavaScript 在运行的时候 始终是 **单线程** 的，不通的语句块是以队列的形式排列在 Eventloop 中安 **顺序执行的**

**思考差异**

本章涵盖许多概念相近，但用途不同的内容，通过比较思考他们的区别和用途以加深理解，比如

+ Debounce vs Throttle
+ 宏任务 vs  微任务
+ 代理模式 Vs 装饰着模式

**动手实践**

在思考比较各种概念后，动手实现一遍，加深理解
